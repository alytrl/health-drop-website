# Health Drop Website

**Live at:** https://thehealthdrop.com

A vibrant, evidence-based health newsletter landing page with newsletter archive.

## Contents

- **index.html** - Landing page with newsletter mockup and signup form
- **archive.html** - Newsletter archive with category filtering
- **style.css** - Responsive design system with vibrant orange/coral branding

## Features

✓ Mobile-responsive design (480px, 768px, 1200px breakpoints)  
✓ Embedded newsletter mockup showing actual product  
✓ "Why Health Drop Is Different" differentiation section  
✓ Professional testimonials from healthcare providers  
✓ Dark mode support  
✓ Accessibility features (WCAG compliant)  
✓ Fast load times (Vercel CDN)

## Local Testing

To test locally before deployment:

```bash
# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment

Deployed via Vercel CI/CD:
1. Push code to GitHub
2. Vercel auto-detects changes
3. Auto-deploys to production
4. Custom domain points to deployment

## Newsletter Archive

Archive page at `/archive.html` shows all newsletter editions with:
- Category filtering (Health, Sleep, Nutrition, Exercise, Mental Health)
- Article metadata (date, read time, tags)
- Email signup form
- Responsive grid layout
